# cypress-javascript-bdd

Pre-Requisites:

      install Node.js 12, 14 or above
      
Clone Repository:

      git clone https://github.com/aatifshehzad/cypress-javascript-bdd.git
      

Setup:

      remove the node_modules folder
      cd to/project/root
      Open terminal and run the following commands:
      npm install
      npx cypress open
      
Run the script:

      cd to/project/root
      npm run test
